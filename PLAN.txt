# План разработки платформы продажи красок (ЛКМ)

## Инфраструктура
[ ] Дизайн-система (палитра индиго/янтарь), шрифты Inter/Montserrat/JetBrains Mono
[ ] Мок-данные: товары, категории, бренды, отзывы, статьи блога
[ ] Типы данных (TypeScript)
[ ] Корзина: React Context + persist
[ ] Общий layout: хедер (sticky), футер, метаданные SEO

## MVP
[ ] Главная (/): hero, категории, хиты, преимущества, лояльность, отзывы
[ ] Каталог (/catalog): сайдбар фильтров, сетка, сортировка
[ ] Карточка товара (/product/[slug]): галерея, табы, калькулятор расхода, похожие
[ ] Корзина (/cart): список, промокод, бонусы, итого
[ ] Оформление (/checkout): контакты/доставка, подтверждение, согласия 152-ФЗ
[ ] Личный кабинет (/account): заказы, бонусы, рефералы, избранное, адреса, профиль
[ ] Доставка (/delivery)
[ ] Контакты (/contacts)
[ ] Юридические (/privacy, /terms, /loyalty-terms)

## Phase 2
[ ] Колеровка (/colormixing)
[ ] Лояльность и рефералы (/loyalty)
[ ] Оптовикам (/wholesale)
[ ] Блог (/blog, /blog/[slug])
[ ] Поиск (/search)

## Проверка
[ ] Проверка в браузере (agent-browser)
[ ] Тесты логики (корзина, бонусы, калькулятор расхода)

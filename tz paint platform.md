# Техническое задание: Платформа для продажи красок

> **Назначение документа:** ТЗ для разработчиков, дизайнеров и нейросети, реализующей платформу.
> **Правовая база:** Законодательство РФ — ФЗ «О защите прав потребителей», ФЗ «О персональных данных» (152-ФЗ), правила дистанционной торговли (ПП РФ № 612), требования Роспотребнадзора к продаже ЛКМ.

---

## 1. Концепция и позиционирование

**Продукт:** B2C/B2B интернет-магазин лакокрасочных материалов (ЛКМ) — для профессиональных мастеров, строительных бригад и DIY-покупателей.

**Ключевые ценности:** Экспертность в выборе краски, скорость получения заказа, доверие через сертификаты и реальные отзывы.

**Монетизация:**
- Прямые продажи (маржа с ЛКМ)
- Оптовые контракты с бригадами и юрлицами
- Платная услуга колеровки
- Программа лояльности с накопительными бонусами
- Реферальная программа для мастеров

**Оплата:** только при получении (наложенный платёж / наличными курьеру / в ПВЗ).

---

## 2. Стиль и дизайн-система

### 2.1 Общая стилистика

**Направление:** Современный индустриальный минимализм. Образ — профессиональный инструмент, не «уютный магазинчик». Чистые блоки, воздух, крупная типографика.

**Настроение:** Уверенность, практичность, экспертность.
**Референсы:** Leroy Merlin (структура каталога), Tikkurila (работа с цветом), Notion (типографика).

---

### 2.2 Цветовая палитра

| Роль | HEX | Применение |
|---|---|---|
| Основной бренд (индиго) | `#1B2D5B` | Хедер, заголовки, основные CTA |
| Акцент (янтарь) | `#F5A623` | Кнопка «В корзину», бейдж «Хит», акции |
| Фон страниц | `#F7F8FA` | Общий фон |
| Фон карточек | `#FFFFFF` | Карточки товаров, формы |
| Текст основной | `#1A1A2E` | Заголовки, основной текст |
| Текст вторичный | `#6B7280` | Подписи, мета |
| Разделители | `#E5E7EB` | Границы, линии |

### 2.3 Типографика

**Основной:** `Inter` — весь UI.
**Акцентный:** `Montserrat` Bold/ExtraBold — крупные заголовки, баннеры.
**Моноширинный:** `JetBrains Mono` — артикулы, номера заказов.

| Стиль | Шрифт | Размер | Вес |
|---|---|---|---|

| H2 Раздел | Montserrat | 32px / 24px моб | 700 |
| H3 Подраздел | Inter | 22px | 600 |
| Body | Inter | 16px | 400 |
| Body Small | Inter | 14px | 400 |
| Caption | Inter | 12px | 400 |

| Артикул | JetBrains Mono | 13px | 400 |

---

### 2.4 UI-компоненты

- **Скругления:** 8px — карточки, 6px — кнопки, 4px — инпуты, 12px — модалки
- **Тени:** `0 2px 8px rgba(0,0,0,0.08)` в покое → `0 8px 24px rgba(0,0,0,0.14)` при hover
- **Анимации:** ease-in-out 200ms (hover), 300ms (модалки)
- **Иконки:** Lucide Icons
- **Сетка:** 12 колонок, max-width 1280px, gutter 24px

## 3. Страницы платформы

---

### 3.1 Главная страница (`/`)

**Цель:** Быстро направить в каталог или на подбор товара. Лаконично, без перегруза.

**Хедер (sticky):**
- Логотип
- Навигация: Каталог / Колеровка / Оптовикам / Доставка
- Поиск, корзина, кнопка «Войти»
- Телефон

**Блоки страницы:**

1. **Hero-баннер** — одна сильная акция или УТП, одна CTA-кнопка «Перейти в каталог»

2. **Популярные категории** (6 карточек с иконкой и названием)
   - Интерьерные краски / Фасадные / Грунтовки / Эмали и лаки / Антикоррозийные / Спецсоставы

3. **Хиты продаж** (горизонтальный скролл, 6–8 карточек товаров)

4. **Преимущества** (4 блока в строку)
   - Сертифицированная продукция
   - Доставка по РФ, оплата при получении
   - Колеровка в любой цвет
   - Бонусы за каждую покупку

5. **Блок программы лояльности и реферальной программы** — краткое описание + CTA «Зарегистрироваться»

6. **Отзывы** (карусель, 3–4 отзыва)

7. **Футер**
   - Навигация
   - Реквизиты юрлица (обязательно по РФ)
   - Политика конфиденциальности, Пользовательское соглашение
   - ВКонтакте, Telegram
   - ИНН/ОГРН, копирайт

---

### 3.2 Каталог (`/catalog`)

**Макет:** Сайдбар с фильтрами (слева) + сетка товаров 3/2/1 колонки.

**Фильтры:**
- Категория
- Бренд (мультивыбор)
- Поверхность применения
- Тип (водоэмульсионная / алкидная / акриловая / эпоксидная)
- Объём (1л / 5л / 10л / 20л / 50л)
- Цена (ползунок от/до)
- Цвет (палитра)
- Только в наличии
- Сортировка: популярное / цена↑↓ / новинки / рейтинг

**Карточка товара:**
- Фото (hover → второе фото)
- Бейдж: Хит / Новинка / Акция / Колеровка
- Название + краткие характеристики
- Цена
- Кнопка «В корзину»
- Избранное (сердечко)

---

### 3.3 Карточка товара (`/product/:slug`)

**Структура:**

1. Хлебные крошки
2. Фотогалерея (основное фото + миниатюры + zoom)
3. **Правый блок:**
   - Название (H1), артикул
   - Рейтинг (звёзды + кол-во отзывов)
   - Цена + цена за литр
   - Выбор расфасовки (кнопки объёма)
   - Выбор цвета / ссылка «Заказать колеровку»
   - Количество + кнопка «В корзину»
   - «Купить в 1 клик»
   - Наличие / сроки
   - Расчёт доставки по индексу
   - Сколько бонусных баллов начислится за этот товар

4. **Табы:**
   - Описание
   - Характеристики (таблица: состав, расход, время высыхания, укрывистость)
   - Инструкция по применению
   - Сертификаты и документы (PDF: сертификат соответствия, паспорт безопасности — обязательно по ГОСТ для ЛКМ)
   - Отзывы (с формой, доступной только подтверждённым покупателям)

5. Калькулятор расхода (площадь м² → литры → банки → цена)
6. Похожие товары
7. С этим покупают (аксессуары: кисти, валики, растворители)

---

### 3.4 Услуга колеровки (`/colormixing`)

1. Hero с заголовком
2. Как это работает (3 шага: выбери базу → укажи код цвета → получи)
3. Выбор палитры: RAL / NCS / Pantone / загрузить фото
4. Калькулятор стоимости
5. Форма заказа
6. FAQ (совместимость, точность, сроки)

---

### 3.5 Корзина (`/cart`)

- Список товаров (фото, название, объём, цвет, кол-во, цена, итого)
- Удалить / перенести в избранное
- Промокод
- Применить бонусные баллы
- Итого: товары / скидка / доставка / сумма
- Кнопка «Оформить заказ»
- «Не забудьте докупить» (сопутствующие товары)

---

### 3.6 Оформление заказа (`/checkout`)

**Шаг 1 — Контакты и доставка:**
- Имя, телефон, email
- Тип: Курьер / ПВЗ (СДЭК, Boxberry, Почта России) / Самовывоз
- Адрес / выбор ПВЗ на карте (Яндекс Карты)
- Желаемое время

**Шаг 2 — Подтверждение:**
- Сводка заказа
- Напоминание: оплата при получении
- Чекбокс согласия с офертой и политикой конфиденциальности (152-ФЗ, обязательно)
- Кнопка «Подтвердить заказ»

> Онлайн-оплата отсутствует. Оплата производится курьеру или в ПВЗ при получении.

---

### 3.7 Личный кабинет (`/account`)

- **Мои заказы** — история, статус, трекинг, повторить заказ
- **Бонусы** — баланс баллов, история начислений, уровень программы лояльности
- **Реферальная программа** — личная ссылка/код, статистика приглашений, заработанные бонусы
- **Избранное**
- **Адреса**
- **Профиль** — имя, телефон, email, дата рождения (скидка в ДР)
- **Мои отзывы**

---

### 3.8 Программа лояльности и реферальная система (`/loyalty`)

**Программа лояльности:**
- За каждую покупку начисляются бонусные баллы (например, 1 балл = 1 ₽)
- Уровни: Стандарт / Серебро / Золото / Платина — зависят от суммы покупок за период
- На каждом уровне повышается % начисления баллов и открываются привилегии (приоритетная обработка заказа, скидка на колеровку и т.д.)
- Баллы можно тратить при оформлении заказа (частично или полностью покрывать стоимость)
- Баллы сгорают через 12 месяцев без покупок

**Реферальная программа:**
- После регистрации каждый пользователь получает персональный реферальный код/ссылку
- Поделился ссылкой → друг зарегистрировался и сделал первый заказ → оба получают бонусные баллы
- Реферер видит в ЛК: сколько друзей пришло, сколько баллов заработано
- Особые условия для мастеров и бригад (повышенный реферальный бонус за оптовые заказы приглашённых)

**Страница `/loyalty`:**
1. Описание уровней программы (таблица привилегий)
2. Как тратить баллы
3. Блок реферальной программы — условия, размер бонуса
4. CTA «Войти и получить свой код»

---

### 3.9 Оптовикам (`/wholesale`)

1. Условия (скидки от объёма, персональный менеджер, отсрочка платежа)
2. Форма заявки (ИНН, название, контакты)
3. Прайс-лист PDF (после заявки)
4. Закрывающие документы: УПД, счёт-фактура

---

### 3.10 Доставка и оплата (`/delivery`)

Обязательный контент по законодательству РФ:
- Способы доставки, сроки, стоимость по регионам
- Оплата: только при получении (наличными / картой курьеру / в ПВЗ)
- Условия возврата (ст. 26.1 ЗоЗПП, ПП № 612)
- Особые условия для ЛКМ (ограничения перевозки опасных грузов)

---

### 3.11 Блог (`/blog`, `/blog/:slug`)

**Рубрики:** Как выбрать краску / Инструкции / Цветовые тренды / Обзоры брендов / FAQ

Внутри статьи — встроенные карточки товаров из материала.

## 4. Технические требования

| Слой | Технология |
|---|---|
| Фронтенд | Next.js 14+ (App Router) + TypeScript |
| UI | Tailwind CSS + shadcn/ui |
| Бэкенд | NestJS или Laravel |
| БД | PostgreSQL |
| Поиск | Typesense или Meilisearch |
| Медиа/CDN | Яндекс Object Storage |
| Карты | Яндекс Карты API |
| Доставка | СДЭК API, Boxberry API, Почта России API |
| Аналитика | Яндекс Метрика |

**SEO:** SSR/SSG для каталога и карточек, Schema.org (Product, BreadcrumbList, Review), ЧПУ, sitemap.xml.

**Требования РФ:**
- SSL/TLS
- Хранение ПД на серверах в РФ (152-ФЗ, ст. 18.1)
- Согласие на cookie при первом посещении
- Обязательные чекбоксы согласия при регистрации и заказе
- Онлайн-касса не требуется при оплате наложенным платежом через СДЭК/Почту — уточнить с бухгалтером схему работы (54-ФЗ)
- Витрина соответствует ПП РФ № 612: полное описание, цена и условия — до оформления

---

## 5. Приоритет разработки

**MVP:** Главная → Каталог → Карточка товара → Корзина → Оформление заказа → ЛК (заказы) → Доставка/Контакты/Юридические страницы

**Phase 2:** Колеровка → Программа лояльности + реферальная система → Оптовикам → Блог → Калькулятор расхода

**Phase 3:** Мобильное приложение → A/B-тестирование → BI-дашборд

---

*Все правовые требования актуальны в соответствии с законодательством Российской Федерации.*
